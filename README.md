# QuantumLeap

The Ultimate Hybrid AI Accelerator Simulator.

This framework is a proof-of-concept for a deep learning framework that uses a "chip" abstraction for computation, incorporates a "quantum-inspired" optimization layer, and includes performance benchmarks against PyTorch and TensorFlow.

## Installation

To install the package in editable mode, run:

```bash
pip install -e .
```

## Benchmarks

To run the performance benchmarks, run:

```bash
python3 benchmarks/speed_test.py
```

## Tests

To run the unit tests, run:

```bash
python3 tests/test_tensor.py
```
